# 计算机基础知识

计算机俗称电脑，是现代一种用于高速计算的电子计算机器，可以进行数值计算、逻辑计算，还具有存储记忆功能。

计算机由 **硬件** 与 **软件** 组成：

- 硬件：看得见摸得着的物理部件；
- 软件：可以指挥硬件工作的指令。

软件的分类：

- 系统软件：Windows、Linux、Android、Harmony（鸿蒙） 等；
- 应用软件：微信、QQ、王者荣耀、哔哩哔哩 等。
